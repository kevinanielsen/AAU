# Exercise 4
Let $w=a^pb^{2p}c$
By the Pumping Lemma, $|xy| \le p$. Because the first $p$ characters of $w$ are all $a$, $y$ must consist of entirely $a$'s.

If we pump the string with $i=4$ $(w=xy^4z)$, we add more $a$ to the front and don't add any $b$. This means that there are will now be more $a$'s than half the number of $b$'s, breaking the rule of $L$ and proving that it is not a regular language.
