# Logic Minimization
Karnaugh map (K-map): a simple and straightforward procedure for minimizing Boolean functions
- K-map is a graphical representation of a truth table
- A diagram made up of $2^n$ squares (cells)
- Each cell represents a minterm of the Boolean function (0 for prime and 1 otherwise)
- Example: a map of two variables (x,y), 4 cells
- The cell value is either 1 or left empty for 0
- We put 1 for each minterm representing the function
## Gray Code
A numbering scheme: 00, 01, 11, 10, where only one variable changes from one word to the next.
# Combinational Circuits
**What is a combinational circuit?**
A circuit of logic gates that performs an operation that can be specified logically by a set of Boolean functions.
The combination circuit consists of an interconnection of logic gates
